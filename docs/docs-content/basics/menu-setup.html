Menu location customization

By default, any options panel created with OptionsKit goes under the "Settings" menu of the WordPress dashboard. If you wish to customize this location you'll need to use the built-in {prefix}_menu filter.

The filters accepts an array with the following parameters:

  • parent
  • page_title
  • menu_title
  • capability

These are described here add_submenu_page.

Note: sample is the slug we've set when we've instantiated the class.

function sample_menu( $menu ) {

    return array(
        'parent'     => 'options-general.php',
        'page_title' => 'My panel',
        'menu_title' => 'My Panel',
        'capability' => 'manage_options',
    );

}
add_filter( 'sample_menu', 'sample_menu' );
