Settings tabs and subsections

In the previous step, you've set a slug for the options panel ( 'sample' ). The prefix of all the filters for your options panel is now sample. We'll prepare some sections for your options panel so we can then add settings.

Adding settings tabs - {prefix}_settings_tabs

The first thing to do is to add a new tab to your options panel. Tabs are required. Through the filter you can add as many tabs as you want. Remember the keys!

function sample_settings_tabs( $tabs ) {
    $tabs = array(
        'welcome' => 'Welcome',
        'another_tab'  => 'Another tab'
    );
    return $tabs;
}
add_filter( 'sample_settings_tabs', 'sample_settings_tabs' );

This will add 2 tabs to your options panel.

Adding subsections - {prefix}_registered_settings_sections

Subsections are not required, but if your panel has a lot of settings, you might want to split them in subsections.

function sample_settings_sections( $sections ) {
    $sections = array(
        'welcome' => array(
            'example' => 'Subsection example'
        ),
    );
    return $sections;
}
add_filter( 'sample_registered_settings_sections', 'sample_settings_sections' );

In the above example, I'm adding a subsection with the key example to the tab with the key welcome.
