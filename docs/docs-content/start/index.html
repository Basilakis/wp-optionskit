Getting Started

Starting a new project

To start a new project, you'll need composer. Add this to your composer.json file:

{
    "require": {
        "alessandrotesoro/wp-optionskit": "dev-master"
    }
}

Then, include the autoloader into your plugin:

require __DIR__ . '/vendor/autoload.php';

Create your options panel

Now that you've included the library into your project you can create your first options panel. To add your panel you need to instantiate the class and add a title to the page.

$sample = new TDP\OptionsKit( 'sample' );
$sample->set_page_title( 'Options Panel' );

The OptionsKit class accepts one argument, a slug. The slug will be used for storing your options and prefix the built-in filters. All your settings will be now saved within the option "sample_settings".

Note: by default the options panel is added under the "Settings" menu of the WordPress dashboard.

Adding action buttons

Action buttons are buttons that will display right next to the options panel title. These can be any type of external or internal links. To add an action button, you need to use the add_action_button method. This method accepts an associative array with title and url as keys.

$sample->add_action_button( array(
    'title' => 'Testing',
    'url'   => '#'
) );

Adding a logo to the panel.

If you wish to add an image right before the options panel title, you need to use the add_image method. Note: there is no css applied to the logo, so you'll need to add your own to make it look good and align with the title.

$sample->add_image( 'url to your image goes here' );
