Vue.js powered options panel for WordPress

WP OptionsKit is a toolkit for developers to create administration options panels for WordPress powered by Vue.js. The UI is consistent with the native WordPress user interface.

Powered by Vue.js

Vue.js is a javascript framework to build interactive user interfaces. Thanks to Vue.js, OptionsKit is capable of providing a fast and interactive interface with no page reload.

Tabbed sections and subsections

Gone are the days of confusing admin panels. Split all your options in sections and subsections to keep your panel nice and easy to use.

Available field types

  • Text input
  • Textarea
  • Select dropdown
  • True/False checkbox
  • Multiple checkboxes
  • Radio buttons
  • Multiselect field
  • Image upload field

Developers friendly API

Creating your own admin panel only takes a couple of lines. If you need more custom fields you can use Vue.js to create custom components, OptionsKit will read them automatically.

An errors handling api is also available to let your users know of any errors with your settings.
